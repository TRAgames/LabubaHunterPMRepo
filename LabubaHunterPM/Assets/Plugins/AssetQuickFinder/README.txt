-------------------------------------------------------------
QUICK START
-------------------------------------------------------------
1 To search owners or dependencies of asset, right click the asset in unity project window, select menu [[Finder] Find Asset Owner].

2 To search asset in current scene, right click the asset in unity project window, select menu [[Finder] Find Asset In Scene].

You have to select at least one asset to open the plugin window.

UPDATING
-------------------------------------------------------------
To update AssetQuickFinder you should preferrably:
- Backup your project.
- Remove previous "Assets/Plugins/AssetQuickFinder" folders.
- Re-import AssetQuickFinder package anew.
